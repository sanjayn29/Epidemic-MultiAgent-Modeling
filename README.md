# EpidemIQs – Epidemic Multi-Agent Modeling UI

> **A collection of two front-end prototypes designed for an Epidemic Multi-Agent Modeling research paper.**

This repository contains **two independent user interface prototypes** created to visualize how an AI-powered multi-agent epidemic research platform could function. These projects are **UI demonstrations only** and are intended to showcase the user experience, workflow, and architecture proposed in the research paper.

> **⚠️ Note:** These projects are **not complete applications**. They do **not** include a production backend, database, authentication service, or real AI agents. All displayed data is mock/demo content used for interface visualization.

---

# Project Overview

This repository includes two separate UI implementations:

| Project | Description |
|---------|-------------|
| **EpidemAI Research Suite** | A research dashboard and control center for epidemic modeling, analytics, simulations, reports, and multi-agent monitoring. |
| **Epidemic Navigator** | An interactive AI research workflow interface that demonstrates how a multi-agent epidemic research pipeline could execute step-by-step. |

Together, these interfaces demonstrate different perspectives of the proposed Epidemic Multi-Agent Modeling platform described in the research paper.

# Important Note

Both projects are **front-end prototypes only**.

These interfaces were developed to demonstrate the proposed architecture and workflow described in the research paper.

The following components are **not implemented**:

- Backend APIs
- Database
- Authentication
- Real AI Agents
- LLM Integration
- Persistent Storage
- Live Epidemic Data
- Job Scheduling
- Multi-Agent Execution Engine

All displayed values, charts, reports, experiments, and agent outputs are **demo/mock data** created solely for UI visualization.
