# Epidemic Navigator

Multi-agent epidemic intelligence platform for outbreak analysis, simulation, forecasting, and research workflows.

## Project Overview

Epidemic Navigator is a React and TypeScript application that helps teams inspect outbreak signals, coordinate analysis workflows, and present epidemic intelligence in a clear dashboard environment.

## Features

- Outbreak-oriented dashboard views
- Multi-agent research workflow pages
- Simulation, forecasting, and analysis entry points
- Modular route-based architecture
- Reusable UI components for operational analytics

## Tech Stack

- React 19
- TypeScript
- TanStack Start and TanStack Router
- Vite
- Tailwind CSS
- Radix UI
- React Query
- React Flow

## Folder Structure

- `src/routes` - file-based application routes and root shell
- `src/components` - shared UI and domain-specific components
- `src/lib` - utilities, themes, and application helpers
- `public` - static assets and browser metadata

## Installation

```bash
bun install
```

## Running Locally

```bash
bun run dev
```

To create a production build:

```bash
bun run build
```
